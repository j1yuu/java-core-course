# Another Library Manager application

Ещё одна Cli-библиотека
Финальный проект модуля "Exceptions"

Используемые навыки:
- Основы синтаксиса Java: классы, методы, условные операторы, циклы.
- Понимание иерархии исключений (checked и unchecked), умение применять:
  - try-catch
  - throw и throws
  - finally
- Умение создавать собственные классы исключений и продумывать их логику.
- Работа с консолью (ввод пользователя) и базовыми коллекциями/массивами, чтобы хранить данные в памяти.
